B WebAPI реализует CRUD операции с базой,в кот.хрянятся данные пользователей и их аватары (PhotoFace). 
А также осуществляет поиск пользователя по файлу его аватара.
WebAPI работает с файлами jpeg.

WebAPI использует LocalDB. При первом запуске будет сгенерирована тестовая база.
(В тестовой базе вместо фото людей загружены фото котиков, "фамилия" котика созвучна с цветом окраса)
Тестовые аватары содержатся в папке Avatars.

В файле WebAPI_Avatars.postman_collection содежится коллекция запросов для PostMan, поясняющая работу WebAPI.
